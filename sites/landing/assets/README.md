# Landing Page 图片资源

请在此目录放置以下图片文件：

## 必需文件

1. **avatar.jpg** - 个人头像图片
   - 建议尺寸：300x300 像素或更大
   - 格式：JPG、PNG 或 WebP
   - 会被裁剪为圆形显示

2. **background.jpg** - 背景图片
   - 建议尺寸：1920x1080 像素或更大
   - 格式：JPG、PNG 或 WebP
   - 会覆盖整个屏幕，建议使用高质量风景或抽象图片

## 注意事项

- 图片会自动添加半透明遮罩，确保文字可读性
- 建议使用压缩后的图片以提升加载速度
- 可以使用 WebP 格式获得更好的压缩效果
